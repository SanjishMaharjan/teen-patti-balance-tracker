# TeenPatti balance tracker

This is a simple web app that tracks the balance of TeenPatti's wallet.

## Getting Started

To get started, you need to clone the repository and install the dependencies using the following command:

```bash
git clone https://github.com/sanjishmaharjan/teen-patti-balance-tracker.git
cd teen-patti-balance-tracker
pnpm install
```

## Usage

To run the app, you need to start the development server using the following command:

```bash
pnpm start
```

This will start the development server and open the app in your default browser. You can access the app at http://localhost:3000.
